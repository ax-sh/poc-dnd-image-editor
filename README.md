# Drag-and-Drop Image Editor (PoC)

A proof-of-concept image editor application with drag-and-drop functionality built with React and TypeScript. This project demonstrates interactive image manipulation using modern web technologies.

## Features

- Drag-and-drop interface for image uploading and placement
- Interactive canvas for image manipulation
- Rendering powered by PixiJS for 2D graphics
- Modern UI built with React and styled with Tailwind CSS
- Fully typed with TypeScript

[//]: # (- and Three.js for 3D elements)

## Technologies

- **Frontend**: React 19, TypeScript
- **Rendering**: PixiJS 8
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Build Tools**: Vite
- **Testing**: Vitest, Testing Library, Storybook
- **Development**: ESLint, TypeScript
